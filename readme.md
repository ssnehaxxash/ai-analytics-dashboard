# AI Analytics Dashboard Suite

A multi-module analytics dashboard built using Streamlit.

## Modules

• Student Performance Analytics  
• AI Resume Analyzer  
• Personal Finance Dashboard  
• Disaster Monitoring Dashboard

## Tech Stack

Python  
Streamlit  
Pandas  
Plotly  
Scikit-learn  
PyPDF2  

## Features

- Interactive dashboards
- Data visualization
- Resume text extraction
- Financial analytics
- Earthquake monitoring map

## How to Run

pip install -r requirements.txt  
streamlit run app.py
